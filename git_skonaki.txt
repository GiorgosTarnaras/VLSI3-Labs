εάν είσαι στο main και θες να προσθεσεις αρχεια :

1) κανεις commit τις αλλαγες tortoise git
2) git push origin main 


εάν υπαρχουν αλλαγες πρωτα τραβας τα καινουργια αρχεια

git pull origin main 


αλλαζεισ branch 


git checkout branch-name

καθαριζεις αρχεια που δεν είναι commit και δημιορυγουν προβλημα στο pull 

git clean -fd



enoneis branch

git merge branch-name
